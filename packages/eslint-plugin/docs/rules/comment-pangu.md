# 规范中文书写，中文，英文和数字之间应该有一个空格

💡 通过编辑器手动修复.

## Fail

```javascript
// loading过程中禁止全选

/**
 * loading过程中禁止全选
 */
```

## Pass

```javascript
// loading 过程中禁止全选

/**
 * loading 过程中禁止全选
 */
```
